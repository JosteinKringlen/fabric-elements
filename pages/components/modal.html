Usually you only want the modal to appear after a certain action has been taken, eg. clicking a button that will delete something. To achieve this you can use the hidden attribute on element, then show the modal at the appropriate time.

Dismissing

The modal can be dismissed by calling hide() on the element.

If the user clicks the backdrops, or hits esc, an event, dismiss, is emitted. You can prevent this event if you don't want to allow the user to cancel the modal until a specific action has been taken.

Events

Name Description
dismiss Emitted when a user presses esc or clicks outside the dialog content. preventDefault() cancels the dismissal.

Methods

Name Description
show Show the modal dialog
hide Hides the modal dialog

Custom Properties

Name Description
